caCSS

CSS in its simplest form.

About

caCSS is a CSS framework. Yeah, right. This is actually just a basic set of CSS styles for you to start working on a web page right away. In other words, this framework thing beautifies the default styles applied by your web browser.

So, what are the features?

  • Lightweight.
  • Simple.
  • No layout or grid stuff. (Do it yourself.)
  • HTML5-ready. (I don't care, I want it now.)

That's all? Yeap.
